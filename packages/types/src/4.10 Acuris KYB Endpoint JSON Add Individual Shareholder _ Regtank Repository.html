4.10 Acuris KYB Endpoint JSON Add Individual Shareholder

The API call to add an individual shareholder’s KYC record to the requested KYB record.

Request Syntax

POST https://{client-portal-server}/v3/kyb/add-individual-shareholder

Common Parameters

  • Request Type: POST

  • Request URL: https://{client-portal-server}/v3/kyb/add-individual-shareholder

  • Request Headers:

    • API Key: <access token>

    • Content Type: application/json; charset=utf-8

Request Body

{
    "requestId": "string",
    "kycId": "string",
    "percentOfShare": "float",
    "remark": "string"
}

Description of the fields in the Request Body

Key
Type
Description

requestId*

String

KYB ID of business to be assessed

kycId*

String

The KYC ID of the associated Individual Shareholders or Ultimate Beneficiaries

percentOfShare

Float

Percentage of shares owned by the individual shareholder in the business entity, represented as a floating-point number (e.g., 25.5 for 25.5%)

remark

String

Additional remarks or supplementary information

* Required field to fill

Response Body

If the action is successful, the service sends back an HTTP Code 200 response.

The following data is returned in JSON format by the service.

{
    "requestId": "string"
}

Description of the fields in the Response Body

Field
Type
Description

requestId

String

KYB ID

Errors

HTTP Code
Description

400

percentOfShare is invalid

400

Add individual shareholder failed. The specified KYB has unresolved actions or has been terminated

400

Record is pending review. Please review and approve the record before making any changes

401

No authentication provided or authentication failed

404

KYB of business to be assessed is not found

404

KYC of individual shareholder is not found.

429

Too many requests — rate limit exceeded

Sample error:

{
    "timestamp": "2025-02-18T07:44:31.738+00:00",
    "status": 400,
    "error": "Bad Request",
    "message": "Add Individual Shareholder Failed. KYB00553 has unresolved actions or has been terminated",
    "path": "/v3/kyb/add-individual-shareholder"
}

Endpoint Example

Example 1:

Successful Endpoint example which returns HTTP Code 200

Request Url

https://companyname-server.regtank.com/v3/kyb/add-individual-shareholder

Request Body

{
    "requestId": "KYB00001",
    "kycId": "KYC00001",
    "percentOfShare": 10,
    "remark": "TEST"
}

Response Body

{
    "requestId": "KYB00001"
}

Example 2:

Error Endpoint example with a non-existent kybId which returns HTTP Code 404

Request Url

https://companyname-server.regtank.com/v3/kyb/add-individual-shareholder

Request Body

{
    "requestId": "KYB00012",
    "kycId": "KYC00002",
    "percentOfShare": 10,
    "remark": "TEST"
}

Response Body

{
    "timestamp": "2025-08-13T04:57:13.581+00:00",
    "status": 404,
    "error": "Not Found",
    "message": "kyb of business to be assessed is not found",
    "path": "/v3/kyb/add-individual-shareholder"
}
