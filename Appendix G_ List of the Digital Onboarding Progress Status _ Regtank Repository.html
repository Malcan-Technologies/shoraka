Appendix G: List of the Digital Onboarding Progress Status

List of status definitions (With IDV function)

Status
Verify Status
Description

URL_GENERATED

In Progress

When the liveness check URL is generated but the end-user has not started the liveness check process

PROCESSING

In Progress

The liveness check URL is generated and the liveness check is incomplete yet

ID_UPLOADED_FAILED

In Progress

The uploaded ID document either fails to meet the required image quality standards or has failed to capture key document data through OCR.

ID_UPLOADED

In Progress

ID document is uploaded successfully

LIVENESS_STARTED

In Progress

The user clicked on “I’m ready” to start the liveness check

LIVENESS_FAILED

Liveness Test Failed

Liveness check & face comparison failed

CAMERA_FAILED

In Progress

The Liveness check started but the camera was not available

EMAIL_SENT

In Progress

The system-generated email with the onboarding link was sent to individual

LIVENESS_PASSED

Liveness Test Passed

Liveness check & face comparison passed (with IDV function)

WAIT_FOR_APPROVAL

Pending Approval

Session pending approval from Regtank system user for sessions (with IDV function)

APPROVED

Approved

The staff has approved this record (with IDV function)

REJECTED

Rejected

The staff has rejected this record (with IDV function)

RESUBMISSION*

In Progress

Resubmission triggered due to document issues, onboarding link to resubmit document to be sent to the individual email address (Can be switched on/off in setting) or resubmit via the original link

EXPIRED

Expired

URL link has expired

The 'RESUBMISSION' status has been discontinued in Version 3.3.0 for new records. However, this status remains functional for existing records created prior to this version
